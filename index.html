About Us

Lithe Editor a free lightweight text editor built for python developers and to help beginners to code. It's easy and fast to use, no need any third party app to run this app!

  • Free and open source project
  • Easy to use and awesome design

This project is open source which means you can get the source code for free at github.

Frequently Asked Questions

Why it takes so long to open?

Lithe editor connects to our services and check for possible updates, infos, and more. It usually takes one to three seconds.

What version is the best to use?

We'd recommend to use the stable version of the app to have a best experience and more. We also working and optimizing the effectiveness and reliability of the app, so stay soon for more information!
